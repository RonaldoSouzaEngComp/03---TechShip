Sistema Web de Ordem de Serviço, está quase 100% finalizado.
Foi desenolvido para se utilizar por uma assistência técnica.
Na tela inicial "Dasboard" temos o número de cliente, OS em aberto, OS concluidas, Receita total
Logo Abaixo o histórico de ordem de serviço criado:
- ID da ordem de serviço;
- Cliente: Nome do Cliente;
- Equipamento: Nome do equipamento;
- Data de estimada de Entrega;
- Status: Aberto, Andamento ou concluido;

No menu "Novo Cliente" você pode cadastrar os dados do cliente e do equipamento na ordem de serviço
No menu "Ordens de Serviço" você pode visualizar todas as OS cadastradas, podendo Editar, Excluir ou criar um PDF e ainda pode exportar as informações para uma planilha do excel.
No menu "Relatório" você pode escolher o intervalo de datas e visualizar "Total de OS no periodo", "Receita no Perído" e um gráfico da receita do período.
No menu ""Configurações" você pode definir o Nome da Empresa, Nome do Técnico na OS e a garantia do serviço em dias.
Pode ser feito o backup das informações do sistema e assim como restaurar essas informações quando necessário.
